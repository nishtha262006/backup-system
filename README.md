# Automated Backup and Restore System

## Description
This project is a Linux-based backup and restore system using Bash scripting.

## Features
- Backup directories with timestamp
- Restore backup files
- Logging system

## How to Run
chmod +x backup.sh restore.sh
./backup.sh
./restore.sh
